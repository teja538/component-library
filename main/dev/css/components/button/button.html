Buttons

The Button component is used to trigger an action or event, such as submitting a form, Add to cart, canceling an action, or performing a delete operation.

Filled Buttons

These are the basic buttons. Use classes btn-danger or btn-success or btn-info or btn-warning for corresponding buttons

Outline Buttons

These are the buttons with outlines. Use classes btn-danger-outline or btn-success-outline or btn-info-outline or btn-warning-outline for corresponding button styles

Button with Icons

You can add corresponding icons along with buttons as follows

Icon Buttons

These are the icon buttons. Add icon-btn class and icon-btn-info or icon-btn-danger or icon-btn-success or icon-btn-warning as per the use case. And if you need ouline icon button then add icon-btn-success-outline class as per the use case.
