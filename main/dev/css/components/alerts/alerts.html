Alerts

An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task. Alert messages can be used to notify the user about something special: danger, success, information or warning.

It's a simple alert box with background filled. Use alert-danger,alert-success,alert-info,alert-warning for corresponding alerts
